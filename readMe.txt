유니티 버전 : 2020.3.6f1 ( -> 2022.3.13f1로 이전 예정 )
유의점 :

 - 바이너리/텍스트 방식 파일 저장

깃허브로 파일관리 시 바이너리 방식은 사람이 바로 읽을 수 없는 방식으로 관리가 어려우므로 기본 저장방법을 텍스트로 변경할 것. (유니티 최신 버전의 기본 세팅)

→ 프로젝트 세팅 → 에디터 → 디펄트 비헤이비어 모드 → 포스 텍스트

 - 메타파일 관리

메타파일이 새로 생성되거나 부재한 경우 Missing 에러가 나타날 수 있으므로 주의할 것.

→ 기본적으로 윈도우 탐색기가 아닌 유니티 프로젝트 윈도우에서 파일 이동할 것, 불가한 경우 메타 파일과 원본 파일을 같이 옮기거나, 유니티 패키지로 만들어 이동시킬 것.
→ 패치버전까지 버전 완전히 통일할 것.
